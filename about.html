About FoodieHub

FoodieHub is a demo food ordering website built with HTML, CSS and JavaScript. It demonstrates a full front-end flow: browsing a menu, adding items to a cart, updating quantities, and placing an order (client-side).

Use this project as a starting point to learn front-end concepts or to connect with a real back-end later.
